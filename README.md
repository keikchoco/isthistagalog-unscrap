# 🍃 Unscrap — Turn kitchen waste into wonder.

Unscrap is an AI-powered smart repurposing platform that transforms domestic organic waste management into an engaging, gamified experience called **The 3R Lab**.

## 🚀 Vision
Built for Filipino households looking for practical, rewarding entry points into sustainable living. Unscrap turns vegetable peels, eggshells, and coffee grounds into resources like compost, natural fertilizers, and DIY cleaners.

## ✨ Core Features

- **AI Scanner**: Snap a photo of kitchen scraps. Powered by Gemini 2.5 Pro Preview, it identifies items, checks safety (texture analysis), and suggests 3R paths.
- **The 3R Lab**: Level up through ranks (Reducer, Reuser, Recycler, Restorer, Zero Waste) and discover materials across 5 rarity tiers.
- **The Bin**: Your personal index of all discovered materials.
- **Scrappy AI**: A conversational eco-companion for scrap tips and sustainability advice.
- **Scrap-Map**: Community integration for local composting hubs and resource sharing.

## 🛠️ Tech Stack

- **Frontend**: React 19 + Next.JS
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **AI**: Gemini 2.5 Pro Preview
- **Backend & Auth**: MongoDB + Clerk
- **Maps**: OpenStreetMap
